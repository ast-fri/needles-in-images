Finding Needles in Images

NiM-Benchmark & Spot-IT for Fine-Grained DocVQA

Parth Thakkar*, Ankush Agarwal*, Prasad Kasu, Pulkit Bansal, Chaitanya Devaguptapu
* Equal contribution
Fujitsu Research India, IIT Patna
GitHub Hugging Face arXiv (Soon)

Abstract

Can MLLMs find tiny, crucial details in dense documents? NiM-Benchmark stresses this ability via tough, fine-grained VQA tasks. Spot-IT enhances model focus through smart patch selection + dynamic Gaussian zoom — improving accuracy significantly over standard pipelines.

Qualitative Examples

Restaurant menu — Spot-IT captures the answer zone precisely.
Lecture screenshot — fine detail localization.
Website UI — relevant region focus.
